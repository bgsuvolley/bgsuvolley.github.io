# BGSU Volleyball — Recruiting Website

A single-page recruiting site for the Bowling Green State University Women's Volleyball program, designed to engage elite prospects and showcase the program's culture, facilities, and staff.

## Overview

- **Stack:** Vanilla HTML + CSS (no build tools or dependencies)
- **Fonts:** Oswald + Source Sans 3 via Google Fonts
- **Images:** Stored in `/images/`
- **Hosting:** Designed for GitHub Pages (static)

## How to Preview Locally

Open `index.html` directly in any browser — no server required.

## How to Publish via GitHub Pages

1. Push this folder to a GitHub repository
2. Go to **Settings → Pages**
3. Under **Source**, select `main` branch and `/ (root)`
4. GitHub will provide a public URL (e.g., `https://username.github.io/repo-name`)

## Project Structure

```
/
├── index.html       # Full site (HTML + CSS + JS in one file)
├── images/          # All photos and graphics
└── README.md
```
